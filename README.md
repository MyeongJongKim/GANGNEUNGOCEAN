# 🌊 GANGNEUNGOCEAN

> 강릉 5개 주요 해변의 **실시간 날씨 · 파고 · 수온** 통합 대시보드

정적 HTML/JS 사이트. 디자인 100% 보존, JavaScript 로직만 Open-Meteo 실시간 데이터로 교체.

## ✨ 기능

- 🏖️ **5개 해변 실시간 대시보드**: 경포해변 · 안목해변 · 강문해변 · 주문진해변 · 금진해변
- 🌊 **해변별 파고 차별화**: 해변마다 앞바다 지점의 Marine 데이터를 개별 조회
- 🚨 **기상 특보 수준 표시**: 풍랑·강풍·폭염·뇌우·이안류 자동 판정 배너 + 해변 카드 경고 배지
- 🌊 **파고 시뮬레이터**: HTML5 Canvas로 실시간 출렁이는 물리 파도 엔진
- 📊 **활동 지수**: 서핑/해수욕 적합도 점수 및 안전 등급
- 📈 **시간별/주간 예보**: Open-Meteo 5일 예보
- 🌡️ **상세 통계**: 풍향·풍속·파주기·수온·습도·조석(다음 만/간조 시각·조위)

## 🔄 데이터 연동

페이지 로드 시 **자동으로** Open-Meteo Weather + Marine API에 연동됩니다. 사용자가 API 키를 입력하거나 데이터 모드를 선택할 필요가 없습니다. 우측 상단 ⚙️ 버튼은 파도 시뮬레이터의 파고/파주기를 잠시 테스트해보는 슬라이더와 수동 새로고침 버튼만 제공합니다.

## 📡 데이터 소스 (Open-Meteo)

- **Weather Forecast API**: `https://api.open-meteo.com/v1/forecast`
  - 기온(2m), 습도, 풍속/풍향(10m), WMO 하늘상태, 강수확률
  - 5일 × 24시간 (정시) + 일별 min/max
- **Marine Weather API**: `https://marine-api.open-meteo.com/v1/marine`
  - 유의파고, 파주기, 파향, 표층 수온, 조위(sea_level_height_msl)
  - 3일 × 24시간 (정시) + 현재값
  - **조석**: 시간별 조위에서 현재 이후 첫 극값(만조/간조)을 찾아 파라볼릭 보간으로 분 단위 시각과 조위(cm, 평균해수면 기준)를 계산해 표시
  - **해변별 개별 조회**: 각 해변 좌표에서 동쪽(외해)으로 0.05° 보정한 앞바다 지점으로 조회해 해변마다 다른 값을 표시. 격자값이 비면 강릉 앞바다 공용 지점으로 폴백

### 기상 특보 수준 자동 판정

기상청 특보 API는 브라우저 CORS 차단으로 직접 연동할 수 없어, Open-Meteo 실황값에 기상청 특보 발표 기준을 준용해 **참고용으로 자동 판정**합니다 (공식 특보 아님, 화면에 명시):

| 항목 | 주의보 수준 | 경보 수준 |
|------|-----------|----------|
| 풍랑 | 유의파고 3m 초과 | 유의파고 5m 초과 |
| 강풍 | 풍속 14m/s 이상 | 풍속 21m/s 이상 |
| 폭염 | 기온 33°C 이상 | 기온 35°C 이상 |
| 뇌우 | WMO 뇌우 코드 감지 시 주의 | - |
| 이안류 | 파고 1.5m↑ & 파주기 8s↑ 주의 | - |

판정 결과는 대시보드 상단 배너와 해변 카드의 ⚠️ 배지로 표시됩니다.

### 왜 Open-Meteo인가?

| 후보 | CORS | 무료 | 해양 | 결론 |
|------|------|------|------|------|
| 공공데이터포털(기상청) | ❌ | ✅ | ✅ | CORS 차폐로 브라우저 직접 호출 불가 |
| KMA API Hub | ❌ | ✅ | ✅ | 동일 CORS 문제 |
| **OpenWeatherMap** | ✅ | ✅ | ❌ (유료 전용) | Marine API는 유료 |
| **Open-Meteo** | ✅ | ✅ | ✅ | **채택** |

Open-Meteo는 비상업용 무료, API key 불필요, CORS 허용, **Marine API 무료 제공**이라는 정적 사이트에 필요한 모든 조건을 충족합니다.

## ⚠️ 알려진 한계

1. **해상 격자 해상도**: Open-Meteo Marine API는 0.08° 해상도. 해변별 앞바다 지점으로 개별 조회해 차별화하지만, 인접한 해변(경포·강문·안목)은 같은 격자에 들어가 비슷한 값이 나올 수 있습니다. 남북으로 떨어진 주문진·금진은 다른 격자 값을 받습니다.
2. **특보는 참고용 자동 판정**: 화면의 특보 표시는 Open-Meteo 실황 기반 자동 판정이며 기상청/해양경찰청 공식 특보가 아닙니다. **실제 입수 결정은 반드시 기상청/해경 공식 정보를 확인**하세요.
3. **비상업용 라이선스**: Open-Meteo는 비상업용 무료입니다. 상업적 사용 시 별도 라이선스 필요.
